# English
(French below)

Besser Pinned Tabs is an essential tool for users who rely on pinned tabs for quick access to their most important websites. This extension safeguards your pinned tabs and intelligently handles new links.

### Background info

The motivation behind writing this extension is that I kept switching back and forth between Chrome/Firefox and Safari or Yandex Browser. These two browsers handle pinned tabs exactly as I want: they protect them against unintentional changes or closure. Clicking on a link to another domain or entering a different URL opens a new tab instead of replacing the page inside the pinned tab. Additionally, CRTL-W (CMD-W) doesn't work on pinned tabs in these browsers. My wish is that Google and Mozilla implement these features directly in the browser code. The extension brings the desired functionality but with some quirks due to technical limitations.

Besser Pinned Tabs does not collect any data and is Open Source Software.

#### Upgrade your pinned tab usage today with Besser Pinned Tabs!
